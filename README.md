# ArtWebCms
-----------

ArtWebCms is a fast, secure, feature rich, scalable Content Management System, powered by ArtWeb using Symfony2 components integrating with the best practices of MVC.


### Requirements
----------------

Requires `PHP 5.3` or greater.
